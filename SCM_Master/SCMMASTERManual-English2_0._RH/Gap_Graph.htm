Gap Graph

On selecting the gap graph display the absolute gap result against each roll number is plotted for all of the sensor positions. Driven rolls are indicated (where applicable) by ‘O’ against the roll number.

The tools button allows the operator to select and deselect which sensor pairs are displayed.

After the selection, the roll gap graph will be displayed showing only the selected gap pairs, in different colours for analysis.

The target gap is shown as a green line and the acceptance threshold levels are shown as red lines. Information such as roll number, drive roll identification and segment breaks are shown at the bottom of the display.

On any of the graphical displays there exists the ability to zoom in on any section of the data. To achieve this position the windows cursor at the top left hand corner of the data to be “magnified” and then dragging the cursor to the bottom right hand corner of the selected data, whilst holding down the left hand mouse button.

To return the zoomed displays to the normal view press the zoom out button.
