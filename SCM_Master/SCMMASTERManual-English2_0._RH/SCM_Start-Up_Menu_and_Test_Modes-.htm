SCM Start-Up Menu and Test Modes

The menu provides a means to select and enter an option.

There is only one available menu in the SCM software, the test menu. The operator moves within the menu by means of the remote control.

Press the On button  to power up the SCM. The SCM will automatically perform a start-up sequence:

1) CHK SD

Checking SD card is present

8) DLOG

Type of system

2) SDHCOK

SDHC card is OK

9) ONBTN

“ON” button used to turn power system

3) FILINT

File System Initialized

10) CHKDIR

Direction sensor check

4) SWVERS

Software version installed

11) SYSCNF

System config OK

5) FWVERS

Firmware version installed

12) USER

You are now on the user menu

6) CPLDV

CPLD version installed

13) CHECK

First option in the user menu

7) XXXXMX

Contract number and Monitor Number

 

 

After the start-up sequence the user will then be placed in the USER menu when it displays CHECK. Press the up or down key on the infra-red  remote control to scroll through the menu options.

To select an option, press the Enter button once .

To Exit an option press the Exit icon .
