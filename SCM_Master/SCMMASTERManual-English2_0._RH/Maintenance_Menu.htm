Maintenance Menu

The following screen gives an overview of the buttons on the Maintenance menu:

The following buttons are displayed in the top-level window:

ASCII Run Data

An ASCII file of any measuring trial can be created using this option. The ASCII file can then be manipulated using Excel or other similar software to create custom displays of the run data. The file type created is a .CSV

Measurement Calibration

Shows the last calibration performed on the SCM i.e. the present state of the calibration. The calibration for individual measuring trails can also be selected and viewed in this display.

Change Language

Allows the user to change the English text on the displays to any available foreign language.

Edit Language

Allows the user to edit certain parts of the text on the measurement display screens.

Edit Target Gap Details

The target gap values for individual rolls can be edited using this display.

Edit Caster Strand Names

The caster strand names can be edited using this display – if applicable.

Edit Run Comments

If required the run comments attached to a measurement can be edited using this option.

Trace Viewer

This is a software tool which can assist the operator in assessing the functionality of the SCM hardware.

Result Colour Edit

This option allows the user to change the colours which are in the measurement display screens.

Printer Test Page Viewer

This option allows the user to view the printer test page after it has been printed with the option below – “Print Test Page”.

Print Test Page

This option prints the test page providing a default printer is set up in Windows. The printed test page can be compared with the one in the “Printer Test Page Viewer”.
