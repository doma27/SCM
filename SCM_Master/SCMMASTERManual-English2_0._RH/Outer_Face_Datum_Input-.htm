Outer Face Datum Input

Outer face datum measurements are the distances between the outer face measuring head flanges and the hard profile of the SCM.

 

The values can be viewed and adjusted by selecting the Outer face datum calibration button on the laptop computer, from the calibration menu.

 

The outer face datum measurement (nominally 1-3mm) is measured after initial manufacture by placing a straight edge across the outer face of the SCM. A gauge is used to measure the distance between this edge and the sensor flange. The result is entered into the computer prior to delivery.

Once the initial values are in place, their accuracy may be checked / altered by studying the initial commissioning measurements, on newly prepared segments only. Due to inherent inaccuracies of the ‘straight edge comparison’ method, any constant error seen on inspection of the caster measurements should be subtracted / added to the datum value in order that the resulting outer profile shows nominally zero.

If the results are + (positive) the value should be added to the datum value at the laptop computer and if the results are - (negative) the value should be subtracted from the datum value.

This value should only be changed if a constant offset is noticed in the results. These results must then be sent to Sarclad for the engineers to review them and guide the customer on the next step.
