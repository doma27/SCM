Gap Sensor Calibration Procedure

  • Prepare the calibration cap and measuring head flanges prior to use, ensuring they are both clean and that the surfaces are smooth and free from damage.

  • Connect the laptop computer to the SCM.

  • Switch the SCM on and ensure the operator Interface displays run.

  • Select the gap calibration button on the laptop computer, from the calibration menu.

There are two different calibration caps. The shallow cap is used for calibration of the outer face sensors; the deep cap is used for calibration of the inner face sensors.

  • Select the desired sensor to be calibrated and press the ENTER key to start the calibration routine.

The gap sensor calibration routine requires that three calibration readings are taken for each sensor.

The three required calibration values are when:

  • The gap sensor is at its rest position.

  • The gap sensor is pressed in with the appropriate calibration cap.

  • The gap sensor is pressed in with the reverse end of the appropriate calibration cap.

 

After the three calibration readings have been taken there is an option to accept or discard the new calibration data.

If the data is acceptable, click the green tick to save the data to the laptop. If the data is not acceptable, click the red cross to discard the calibration.

If the data is accepted it will be saved onto the laptop PC.

 

Acceptable new calibrations are highlighted in blue, failed calibrations are highlighted in red.

Important Warning Messages

In order to save the calibration data to the SCM you need to select the calibration transfer button before exiting the calibration menu.

Failure to do this could result in loss of the new calibration data.

 

When connecting to the SCM for calibration the operator interface will automatically display “CALIBR”.

If for any reason the “EXIT” button is pressed, saving the calibration data may not be possible.
Pressing this button in the calibration screen will re-establish communication and saving will now be possible.

 

The gap sensors have a linear characteristic and the response of each device is displayed by the graph on the right hand side of the calibration screen.
