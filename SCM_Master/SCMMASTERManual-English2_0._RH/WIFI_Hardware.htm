WIFI Hardware

 

Figure 9: MOXA Router

 

WIRELESS ACCESS POINT (ROUTER)

 

This is the MOXA router (access point).

It is typically installed at a half-way point between the SCM and the most common point for data download, for example on the casting floor or main control room area.

The unit receives the wireless transmission data from the MOXA NPort and relays it directly to the Laptop Computer.

Figure 10: MOXA Nport Unit

 

WIRELESS SERIAL DEVICE SERVER (NPort)

This MOXA NPort unit is installed in the SCM in the Wi-Fi Enclosure.

The unit converts the RS-232 signal coming into the serial port from the Single Board Computer (SBC Single-board computer) into a wireless transmission which is sent to the MOXA Router (above).

Figure 11: Wi-Fi Aerial

 

WIFI AERIAL

The Wi-Fi aerial is mounted inside an acetyl resin block which is potted and then mounted on top of the SCM.

This aerial is then wired directly into the Wi-Fi enclosure.
