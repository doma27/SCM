INC - Individual Inclinometer Devices that measure angles relative to the horizontal Transducer Test Option

Selection of the  Inc   option from the test menu allows the user to select and view the output of individual analogue sensors. On entering the option, if the calibration is good, the display will say the sensor number will be briefly displayed, followed by an indication of the sensor type and location on the SCM body, followed by the output for that sensor, e.g.

 calok – Calibration ok

 tx034    – Sensor number 34

 lsin      – Left Left when viewed from the mould in the casting direction. side inclinometer

89.9   – Output value of the sensor in degrees

If the sensor blade is pressed in the output value should change.

To view the output of a different sensor, press the arrow buttons on the remote control.
