PRINCIPLES OF MEASUREMENT

Roll Gap

The gap measurements are collected from the inner and outer face sensors as they pass the inner and outer face rolls whilst the SCM runs in a stable manner through the caster.

Figure 12: Roll Gap Measurement Diagram

The outer face sensors are used to ensure correct roll counting and measurement referencing.

The SCM is mechanically designed so that the sensors run in a stable, straight line across the roll as it is being measured. The data from the inner and outer sensors is stored in computer memory when the gap measured is a minimum value.

The stored data is converted to a physical distance for each sensor by comparing to the sensorâ€™s calibration, so determining the exact length each sensor has been depressed at that point.

The SCM flange-to-flange thickness for each sensor position is added together with a calculated offset (to compensate for the attitude of the SCM in the caster) to form a total gap reading.

The resultant roll gap is compared with the theoretical target roll gap and is displayed as an error compared to this theoretical target.
