Results are Unacceptable

If results are grossly incompatible with the assumed condition of the caster, the most likely cause is one of the following:

  • The sensor calibration has been carried out incorrectly.

  • The SCM flange-to-flange thickness has been measured incorrectly.

  • A sensor has become faulty, in which case the particular sensor which shows faulty results must be verified as functional and if not, replaced and re-calibrated. The SCM thickness must also be verified.
