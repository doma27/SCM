System Overview

The SCM electrical system consists of a main data logger unit, a number of M8 distribution blocks and the sensors. The outputs from the sensors are fed into the distribution blocks and then into the data logger unit. The data logger samples the signals and stores the information in a run file on the SDHC memory card, which can then be downloaded to the laptop computer.

Commands are sent to the SCM via the infra-red remote control handset. The Infra-red commands are received by the operator interface. As well as receiving commands the operator interface also has an LED display and indicators to provide information to the operator regarding the status of the SCM.

The internal SCM batteries are charged by an external battery charger mounted on the SCM stand. The connection from the battery charger is by a cable and plug which connect to a socket mounted on the SCM body. The same socket is also used for downloading data from the SCM to the laptop computer via the data transfer cable.
