If the inclinometers are removed from their mounting position, they need to be re-fitted correctly in order to achieve good calibration.

The inclinometer output is in the range of 0 – 5v and the voltage change depends on the angle of the device. This is converted into a digital value in the range of 0 – 32000.

 

By adjusting the position of the inclinometer, the maximum and minimum values can be set to give the widest range of output values.

The suggested way to set these sensors is to take a calibration measurement at -2° which should be between 50 and 200 counts.

Another measurement should be taken at 92°. This reading should be <32000. The table on the previous page will give approximate values for both -2° and 92°.

The values shown previously are not absolute, as they will vary between devices. The important part is the relative change in output between the angles and that both ends of the angular scale fall within the device’s maximum and minimum output values.

These devices are set up during commissioning and should not need to be adjusted. However, if a device becomes faulty and has to be replaced, then the above procedure should be followed.
