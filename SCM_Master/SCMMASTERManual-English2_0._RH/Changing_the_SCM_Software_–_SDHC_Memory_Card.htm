Changing the SCM Software – SDHC Memory Card

1.    Disconnect the battery and charger.

2.    Remove the electronic enclosure cover plate to enable access to the SCM computer.

3.    Open the electronic enclosure and identify the SDHC Memory Card (shown below).

4.    Carefully extract the old SDHC memory card and replace with the new card ensuring correct orientation.

5.    Close the electronic enclosure ensuring a watertight seal and finally replace the cover plate.

6.    Re-connect the battery and charger.

The SCM is now ready for use.
