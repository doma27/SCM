Making a Dummy Measurement

To make a dummy measurement the SCM should be in the maintenance stand. Then power up the SCM, when the SCM is powered up ‘CHECK’ will appear on the SCM display.

Then press the record button on the remote control, the SCM will then start recording.

The operator should now go around the SCM and depress each sensor in turn 3 times, using the 12mm calibration cap.

After you have depressed all sensors, press the Stop button on the remote control. The recorded data should then be downloaded to the SCM laptop.

When downloading a ‘dummy Measurement’ an ‘X’ will appear in the download screen, this is because there is not enough data to be seen as a caster measurement; data should be labelled as a dummy run, and the run number should be noted.
