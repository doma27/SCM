Interface Units

There are two interface units. One fitted to the nose and one fitted on the side face of the SCM.

They are interchangeable and identical.

Both interface units respond to infrared remote control commands.

The items are encapsulated within an acetyl holder.

The unit consists of:

  • An infra-red receiver to detect and amplify coded signal from the remote control.

  • A four-digit display unit to show the state of the SCM.

  • Several LEDS showing SCM Data Logger activity at switch on/off and Hydraulic system status if installed.
