Setup of Client PCs for Viewing Data through a Network

On both the Server and client PCs the folder and file structure should be as follows:

  • From the root directory C:\  there needs to be the folder SCM.041

  • Within this folder there should be the sub folders of the Sarclad software and one Sarclad executable program.

  • The PC executable, “SCM041.exe” is the main Sarclad PC program.
