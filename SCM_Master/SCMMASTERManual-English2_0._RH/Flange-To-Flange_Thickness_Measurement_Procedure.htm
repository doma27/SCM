Flange-To-Flange Thickness The distance between the flanges of the inner and outer face gap sensors. Measurement Procedure

 

The SCMÂ flange-to-flange thickness is defined as the distance between the gap sensor Sensors that are used to measure the roll gaps, and outer roll condition flanges of the inner and outer faces of the SCM. Each gap sensor flange has a hole into which the DDG rods should be inserted through the SCM to the opposite sensor flange.

 

The thickness of the SCM is measured between sensor pairs on the inner and outer face The face of the SCM that is in contact with the Outer Face Rolls, may also be referred to as fixed or back face..

When setting up the DDG, the use of a calibration sleeve is required. This is normally wider than the SCM body. In addition to the sleeve, a gauging ring/s is also used.

The gauging sleeve is always larger than the SCM body as the gauging ring has to be deducted from the total sleeve length value.
The value of the SCM gauging sleeve is the thickness of the SCM plus the ring thickness.

      =          -    

Pre-set Value    =          Sleeve Length       -       Gauging Ring

The purpose of the gauging ring is to elevate the depth gauge from the flange surface so the DDG can sit flush, otherwise the DDG would not reach over the lip between the body and flange surface.

Using the example above, the pre-set value is then inputted into the DDG and placed inside the sleeve. The 236mm sleeve length is then set to 218mm on the DDG and set in reverse mode.

The reason it is done like this is because the ring adds 18mm onto the total length of the body, so if you were doing a measurement of the body without the pre-set value being set, the total value would read approx. 236mm.

This value, if inputted into the software would be wrong as it is 18mm greater. The way around this is to set the sleeve length with this value already deducted.

The SCM body is susceptible to rolling from prolonged use and/or operator error so the DDG must me set in reverse mode to show any negative deviations from when first supplied new.

The sleeve length is determined by the thickness of the SCM being supplied and the length of DDG rods available.
