Gap Error Measurement Comparison

The gap error measurement comparison display allows the operator to compare gap error data for up to five different measuring trials.

The measurements are selected using the load measurement button, with a different colour line representing each trial.

Each line on the display is the average result of all of the gap errors at each roll position.

The tools button allows the operator to select which gap sensor Sensors that are used to measure the roll gaps, and outer roll condition pairs are used in the calculation to determine the average error value.

Behind the individual average lines, the yellow shaded area indicates the range of error values of the selected gap sensor pairs.

Roll information such as roll number, drive roll identification and segment breaks are shown at the bottom of the display.
