Infrared Remote Control

The remote control (typical unit shown) is a hand held unit used to transmit up to twelve commands at the press of the appropriate button. The unit is battery powered (9v alkaline size PP3) and shows an illuminated red LED when data is being transmitted.

START DATA LOGGING

Used at the start of a measuring trial to initiate data logging of the SCM. Button has to be pressed and held for three seconds.

STOP DATA LOGGING

Used at the end of a measuring trial to end data logging of the SCM. Button has to be pressed and held for three seconds.
