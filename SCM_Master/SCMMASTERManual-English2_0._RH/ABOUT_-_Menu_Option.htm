ABOUT - Menu Option

Selecting this option in the user menu will take you to another menu which has the following information:

 swtype   - Displays the software type (typically: DLOG)

 swvers   - Displays the version of software which is installed

 swdate   - Displays the date when the software was installed

 contra   - Displays the machines individual contract number (XXXX)

 monitr   - Displays the monitor number

 time    - Displays the current time

 fwvers   - Displays the version of firmware which is installed

Pressing  on each option will display the information for that option.
