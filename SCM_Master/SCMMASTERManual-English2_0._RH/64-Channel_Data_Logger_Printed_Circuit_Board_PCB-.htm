64-Channel Data Logger Printed Circuit Board (PCB)

This card executes the computer program and controls all aspect of communications, data storage, Infra-red reception and decoding.

Figure 32: 64-bit Data Logger  PCB

What this PCB does:

  • Display transmission and control.

  • Data Communication interfacing.

  • Interfacing of the tilt switches for the automatic power on and stop record functions.

  • A four hour hardware power down function to help preserve battery life.

  • Temperature and battery voltage monitoring and low voltage shut down.

  • Receiving the analogue signals from the gap and inclinometer sensors and performs a 16-bit conversion into digital signals.

  • This card also receive direct digital inputs

  • The unit provides the Roll Rotation and Roll Detection sensors with either a +12V or +5V supply depending on the type of sensor used. The unit accepts the output from the Roll Rotation and Roll Detection sensors and provides buffering between the sensor outputs and the Data Logger digital inputs. Providing the data logger digital inputs with either a 0V or 5V discrete voltage corresponding to a logic level 0 or logic level 1.

  • The unit provides the sensors with a +12V supply and accepts the output from the water spray accelerometer sensors. This signal is then filtered and conditioned to give a 0V to 10V analogue signal

On board there is memory storage in the form of a removable SDHC card which stores the measurement data, calibration data and customer specific software.

The single board computer provides the real time clock function which is backed up by its own separate miniature battery which is the only serviceable component on the board.

Located in the corner of the card is a toggle switch which can be used for forcing the power supply card to boot in the event that there is a problem with the infra-red transmitter or receiver components.
