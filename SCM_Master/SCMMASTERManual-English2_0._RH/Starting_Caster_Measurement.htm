Starting Caster Measurement 

Switch on the SCM  and allow the system to execute its start-up sequence, (this may take up to 30 seconds).

The operator will now be placed in the USER menu and check will be shown:

 check 

Press the record button  on the remote control for 3 seconds to start data logging.

The display will show the following sequence:

 run     

Run started

 loging  

Logging information

The remaining memory time will count down each second until zero minutes and seconds, at which time data collection will cease.

The SCM should now be passed through the caster at a constant speed. The speed should be less than 1.5m/min. This is required so the data is collected with the greatest accuracy.

Remember to sequence the drive rollers as the SCM passes between.

The SCM can be stopped at any position within the caster and its direction reversed without the need to reset the SCM computer. However, only one change of direction should be performed per measuring cycle.

If multiple direction changes are required then the SCM should be reset manually by pressing the stop record button  on the remote control for 3 seconds.

The data is now safely stored in memory and will not be lost if the SCM is powered off.

i
 Important Information

Do not power the SCM off until the save sequence has been completed and the storage location number displayed, otherwise the data will be lost and the caster measurement will have to be repeated.

In total 5 different measuring trials can be stored in the SCM at any time. A measurement which has involved a change of direction and so has up and down data present will be still saved as a single file.

Once the SCM has finished the caster measurement, it should be removed from the dummy bar chain in the usual manner.

At the end of the measurement sequence the recorded data can be transferred to the laptop computer.
