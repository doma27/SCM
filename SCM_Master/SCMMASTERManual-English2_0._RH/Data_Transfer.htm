Data Transfer

Switch on the laptop computer and start up the SCM program.

Connect the laptop computer COM 1 port to the SCM Com / Charger socket using the communication cable provided.

 

Select the cable data transfer option button from the top level window on the main computer.

 

The laptop will communicate with the SCM to determine if there are any new measurements to download.

During the start of the communications the software looks for an available communications port.

If a cable is connected it will automatically select the cable port.

If Wi-Fi is being used, it will automatically select the Wi-Fi port.

Data is exchanged between the laptop and the SCM. Progress of this is displayed on the left hand side of the data transfer screen.

The icons are provided in order from the top:

§  Comms port established

§  Contract number established

§  Monitor number established

§  Data established

§  Downloading started

Following successful initial communications, the SCM will be interrogated to identify what measurements are available in memory for transfer.

The SCM has five memory locations in which data is stored. The date and time the data was stored to memory are displayed adjacent to each memory location.

If measurement data is identified the program will indicate to the operator which of the five files contain data. The available files are highlighted by a yellow border around the selection buttons.

In order to download data from any memory location the corresponding button must be pressed. This does not necessarily have to be the most recently performed measurement.

On pressing the desired button downloading of the data from that memory location commences.

When the SCM has finished transferring the data a small tick is displayed to indicate that the transfer is complete and that it has been successful. Total transfer time will take approximately two minutes for cable/WIFI, depending on the file size.

The operator is then prompted to select the appropriate caster and strand on which the measurements have been taken, and to enter any necessary run comments.

NOTE: Before you can exit this screen, you need to input at least three characters in the Run comments box.

 

After the appropriate selection and comments have been entered, click the Exit button, which will be flashing, to proceed to data analysis.

Data analysis commences automatically on exiting the comment entry screen by pressing the button.

During data analysis the data collected from the caster is reviewed to ensure that it is valid data and that it is corrected allocated to the rolls of the caster.

The progress through the stages of analysis is reflected by the progress bars on screen.

If the analysis is successful the progress bar will be at 100% and a small tick will be present along with an arrow to indicate the detected direction of measurement through the casting machine.

If the analysis is not successful the progress bar will halt and display a cross symbol.


If this occurs there could be a problem with the measurement performed or with the SCM hardware, in which case the inclinometers should be checked and the measurement repeated. If the problems persist then contact Sarclad UK for further assistance.

In the event that the measurement cannot be analysed the data will not be available for viewing. If this occurs, the calibration of the body-mounted inclinometer Devices that measure angles relative to the horizontal should be checked and the measurement repeated. If the problems persist, then contact Sarclad UK for further assistance.

Whether the analysis is successful or not the data will still be saved and allocated an automatic run number. The run number along with the entered run comments is displayed beneath the analysis information.

 

The display will then revert back to the main transfer screen and highlight the next most recent measurement available for transfer by flashing the button yellow. If required all of the available measurements can be transferred following the same procedure as above, else the screen can be exit by selecting the exit button.

In the event of a communication error, a message will be given and the procedure should be repeated.
