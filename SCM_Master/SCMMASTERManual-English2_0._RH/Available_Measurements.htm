Available Measurements

The following buttons are displayed when clicking the Available Measurements menu:

Roll Gap Results

Outer Roll Condition Results

Roll Bend Results - Optional

Outer Roll Alignment Results

Roll Rotation Results - Optional

Water Spray Results - Optional

Battery Status and Temperature Graph

The following additional options are available on the button bar:

 

Print Screen

Prints one page of result data of the current display being viewed.

Print All

Prints the results for all measured rolls.

Print Selection

Opens an additional window where each display type for each measurement type can be selected for printing.

Load Measuring Trial

Selects the measuring trial to be displayed and up to four additional trials for data comparison.

Zoom Out

On any of the graphical displays there exists the ability to zoom in on any section of the data. The zoom out button is used to return the display to the normal view.

Tools

For certain result displays, the tools button is not visible whereas for others it is available (for example, for graph and measurement comparison displays) and allows the operator to select different display options relevant to the results being viewed.

Over Threshold Summary

This button allows the operator to obtain an overview of every roll for every available measurement type.

Exit to Top Level Window

The exit button will return the software to the top-level window display page when selected.
