Over Threshold Summary Button

This button allows the operator to obtain an overview of every roll for every available measurement type.

After you have clicked the button, a summary table appears which, on the left hand side of the table, indicates the individual roll positions. Across the top of the table, the available measurements are displayed and in columns beneath each measurement a cross or tick indicate whether the data for that measurement, at that roll position, is within the set tolerance limits.

Data within the threshold limits will be indicated with a tick.

Data outside of the limits will be indicated with a cross.

All of the sensor data for each particular measurement type has to be within tolerance before a tick is displayed.

Using the cursor to select any tick or cross on the table will change the PC screen to display the selected measurement type at the selected roll number.

To close the threshold summary table select the exit button.
