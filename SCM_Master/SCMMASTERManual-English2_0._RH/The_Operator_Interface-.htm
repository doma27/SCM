The Operator Interface

This is a 14-segment “starburst” display with 6 digits and a decimal point that separates each digit from the last. There are also six additional LED indicators that provide information regarding the status of the SCM.

Characters shown on this display will appear as follows:

 

The LED Indicators represent the following:

 

LED

Indication

Condition

Red, Static

Indicates when a command is received from the remote control

Red, Flashing

Indicates when the solenoid value is energised.

(Only used on Hydraulic systems)

Yellow, Static

Indicates when the SCM has powered off due to a low battery voltage

Red, Static

Indicates when the SCM is powering down.

Green, Static

Indicates when the SCM is powering on and is running.

Red, Static

Indicates when the hydraulic system is at the correct pressure.

(Only used on Hydraulic systems)
