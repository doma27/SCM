Trace Viewer

The trace viewer is a software tool which can assist the operator in assessing the functionality of the SCM hardware. The trace viewer will allow the operator to see a trace graph for each sensor, and will help the operator to decide whether a sensor needs to be replaced.

The trace viewer will show any signs of excessive electrical noise and will help to determine whether a problem exists with the sensor, signal conditioning card or the power supply (DC-DC Converter/PSU).

 

To use the trace viewer:

  • Select the trace viewer icon from the tools menu.

 

  • Then select the green filing cabinet icon.


  • Then select the data to be viewed.

  • When the measurement is loaded into the trace viewer, the operator will see a list of sensors down the right hand side of the screen.

  • Click each sensor in turn to view the trace in the main trace window.

NOTE: The important observation here is the “rest” level. The rest level is the steady state of a fully extended gap sensor; this is indicated by the highest parts of the trace.

 

In this example, we can see that the “rest” level is clean. The trace below shows TX 1 is good and that the top of the trace has a clean straight line indicating that there is no excessive electrical noise.

NOTE: By dragging the cursor from top left to bottom right, you can create a window that zooms in on the appropriate are of the graph. See below for an example of the zoomed area.

The values along the bottom of the trace screen (X-axis) shows time, in computer counts, and the values on the left side of the screen (Y-axis) shows the digital representation of the analogue sensor output in computer counts in relation to the amount the sensor has been depressed.

 

The trace below shows a measurement that has excessive electrical noise on TX 8.

If the noise is present on all sensors, then you need to investigate the power supply which will be either a DC-DC converter or a PSU.

If the noise is only present on one sensor, then that sensor should be replaced and a ‘dummy’ measurement carried out in the SCM stand to confirm the problem is solved (this will be explained later).

If the problem is still present after the sensor has been replaced then this will indicate that the problem is likely to be on the signal conditioning card.

Before replacing sensors and/or electronic cards, all cables, connecters and terminals should be inspected for signs of damage, and loose connections should be tightened.
