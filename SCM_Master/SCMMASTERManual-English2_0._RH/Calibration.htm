Calibration

Connect the laptop computer to the SCM.

Select the inclinometer Devices that measure angles relative to the horizontal calibration button on the laptop computer, from the calibration menu.

 

1.     1.    Select the desired inclinometer to calibrate. This can be done by checking or un-checking the box above each inclinometer column . In a typical scenario, you would select all of them.

2.     2.    If the rotary table has been set up correctly, as previously described, a half turn, anticlockwise, of the wheel should line up the number 2 on the Vernier scale with the zero on the rotary table scale.

3.     3.    Power on the digital spirit level - the digital display should indicate -2°.

4.     4.    Press the ENTER key on the laptop to take a reading for the -2° point.

5.     5.    The display will automatically advance the highlight cursor to the next calibration value.

6.     6.    Now rotate the wheel clockwise so that the 0 on the wheel scale, lines up with the zero on the rotary table scale. At this point the spirit level should read 0°. This should continue to 92° rotating the wheel clockwise, lining up the 0 or 2 on the wheel with the zero on the rotary table.

7.     7.    Each half turn of the wheel will rotate the table 2°.

8.     8.    Rotate the rotary table in steps of two degrees from -2° to 92° collecting data into the laptop computer memory at each step. This should be done using the Vernier scales as they are more accurate than the spirit level as the measurements move towards 92°.

Because the digital spirit level can only read 0 to 90 degrees, 92° would be displayed as 88° when rotating in a clockwise direction.

NOTE: To eliminate the effect of backlash / hysteresis always approach all the angular positions to be calibrated from the same clockwise direction of rotary travel.

Acceptable new calibration readings will be highlighted in blue. Failed readings will be highlighted in red.

As the new calibration values are taken the data is temporarily saved on the laptop PC.

 

In order to permanently save the data to the SCM it is necessary to select the calibration transfer button before exiting the calibration menu. Failure to do this will result in loss of the new calibration data.

 

At the end of the calibration re-build the inclinometer assembly, ensuring that the blade moves smoothly on the springs and spacers.
