Gap Measurement Comparison

The gap measurement comparison display allows the operator to compare gap data of up to five different measurements.

The measurements are selected using the load measurement button, and a different colour line represents each trial.

Each line on the display is the average result of all of the gap readings in that measurements at each roll position.

The tools button allows the operator to select which gap sensor pairs are used in the calculation to determine the average value.

Behind the individual average lines, the yellow shaded area indicates the range of values of the selected gap sensor pairs.

Roll information such as roll number, drive roll identification and segment breaks are shown at the bottom of the display.
