Configuration of the Server PC

To view the latest measuring trials that have been downloaded to the server PC it is necessary to configure the client PCs so that they refer not to their local run data directory but to the directory of the server PC.

Through Windows navigate to the directory C:\SCM.041\config\, and open the Network.INI file using Notepad or similar text editor program.

On the Server PC the file should read:

[Network]

Drive=C

Enabled=0

NetworkUNCUsed=0

UNC=0

On the Client PCs the file should read:

[Network]

Drive=M

Enabled=1

NetworkUNCUsed=0

UNC=0

After editing the file as required, save and exit the text editor program.

On the Client PCs

Using Windows:

§  Open:       My Computer

§  Tools menu

§  Map Network Drive

Select the desired drive letter. E.g. drive M.

(The use of drive M: is only an example). The drive letter to use is the drive letter the client machine is using to see the SCM.041 folder on the Server PC.

Using the Browse option navigate through the network to the server PC’s “SCM.041” directory and then finish.

In the event that either the folder is not accessible because of accessibility or user rights issues contact the local network administrator to obtain the necessary privileges.

Once configured correctly, when the Sarclad PC program is started on the clients PC, the program will refer to the server PC to view the latest measuring trials that have been performed.

i
 Important Information

The server PC must always remain powered on so that the necessary files can be accessed by the client PCs.
