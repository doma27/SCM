Configuration of the Server PC

To view the latest measuring trials that have been downloaded to the server PC it is necessary to configure the client PCs so that they refer not to their local run data directory but to the directory of the server PC.

Through Windows navigate to the directory C:\SCM.041\config\, and open the Network.INI file using Notepad or similar text editor program.

On the Server PC the file should read:

[Network]

Drive=C

Enabled=0

NetworkUNCUsed=0

UNC=0

On the Client PCs the file should read:

[Network]

Drive=M

Enabled=1

NetworkUNCUsed=0

UNC=0

After editing the file as required, save and exit the text editor program.

On the Client PCs

Using Windows:

  • Open:       My Computer

  • Tools menu

  • Map Network Drive

Select the desired drive letter. E.g. drive M.

(The use of drive M: is only an example). The drive letter to use is the drive letter the client machine is using to see the SCM.041 folder on the Server PC.

Using the Browse option navigate through the network to the server PC’s “SCM.041” directory and then finish.

In the event that either the folder is not accessible because of accessibility or user rights issues contact the local network administrator to obtain the necessary privileges.

Once configured correctly, when the Sarclad PC program is started on the clients PC, the program will refer to the server PC to view the latest measuring trials that have been performed.

DETAILED CASTER MEASUREMENT SEQUENCE

A caster measurement is where the SCM is installed on to the dummy bar chain in place of the dummy bar head. It is activated and then inserted in to the caster to the desired reversing point, storing data as it travels through the caster.

Important Warning Messages

Before attempting to insert the SCM into the caster it is first essential that the caster rolls are at the correct target gap. Failure to ensure this could result in serious damage being sustained to the SCM body.

Driven and moveable caster rolls should be sequenced so that they apply no load onto the SCM body as it passes through the rolls during caster measurement.

Under no circumstances should any attempt be made to move the SCM upward into the mould further than the uppermost position, since this will cause damage to the SCM and to the copper mould.

The connector on the SCM body must have its protective water sealing cap fitted before the SCM is used in the caster.

Before lifting the SCM with the crane, ensure that the lifting lugs are in good condition and that the brakes are applied to the stand wheels. Take care when lifting and ensure the transfer of weight causes no movement of the storage stand.

Take due care when moving the SCM on its stand. Heavy weights are involved in this procedure. At least two operators are required

The internal electronics should not exceed a temperature of 70 degrees centigrade.

The caster must be cold or the water sprays must be functioning before the SCM may be inserted into the caster.

The maximum speed of caster measurement is 1.5 meters per minute.
