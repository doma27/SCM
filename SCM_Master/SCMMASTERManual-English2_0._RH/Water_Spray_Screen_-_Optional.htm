Water Spray Screen  - Optional

This is an optional screen that shows the relative amount of water spray detected by the water spray sensors on the inner and outer face The face of the SCM that is in contact with the Outer Face Rolls, may also be referred to as fixed or back face. at positions across the caster width.

The number and location of the water spray nozzles is shown at each roll position.

An individual value and corresponding graph is displayed for each sensor position across the roll width.

 

 

If no water spray is detected, the result will be 0, up to a maximum of 100% if high spray is detected.

The tools option is available and is used to set the desired adjustment of the individual water spray loops. In general, this value will be set during the installation period and should not be adjusted unless there is a change in the water spray system within the caster.

Pressing the up or down buttons for each zone sets the level of sensor sensitivity for that zone - the greater the value, the higher the sensitivity.

The value can be set between 0 to 50 in 0.05 increments.
