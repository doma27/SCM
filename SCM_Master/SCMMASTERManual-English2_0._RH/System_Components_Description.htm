System Components Description

Figure 1: Strand Casting Monitor (SCM) on stand

The SCM is custom designed for each individual casting machine. The main body and the ancillary equipment are all fabricated from steel, which is nickel plated to resist corrosion. The SCM is robustly designed to endure the harsh environment in which it has to operate. Mechanically it can withstand occasional hydraulic loading from driven rolls present within a caster, though regular hydraulic loading will damage the equipment.
